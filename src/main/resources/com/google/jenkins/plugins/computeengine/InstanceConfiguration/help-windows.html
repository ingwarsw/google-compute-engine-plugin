Enable this option if the launched machine is a Windows VM. The Windows VM must have the OpenSSH server feature installed on it, and you must have configured an administrative user with a username and password.

The configured username and password should be set in the "Windows Username" and "Windows Password" fields.

You can use the Startup Script to automatically install the OpenSSH server role and set up a user with:

Add-WindowsCapability -Online -Name OpenSSH.Server~~~~0.0.1.0
net user /add Build password
net localgroup administrators Build /add

See the Windows VM Instances documentation for more information.
